# teXbuilder
A simple cli / tui tool for creating and compiling LaTeX documents.

## Requirements
Requires linux with a version of texlive installed.

## Installation
Clone the project using `git clone --depth=1 https://github.com/linc-reil/teXbuilder.git` and compile the project using `cargo build --release` or run the installation script (which installs the built binary in `~/.local/bin/` under the name `texbuilder`).
